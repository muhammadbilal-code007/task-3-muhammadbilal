Recommendation System

A simple command-line Recommendation System built with Python. Users can choose Movies, Books, or Games, select a genre, and get recommendations.

Features

- Movie recommendations
- Book recommendations
- Game recommendations
- Genre-based search
- Interactive menu
- Input validation

Technologies

- Python 3
- Functions
- Dictionaries
- Loops
- Conditional Statements

How to Run

bash
python recommendation_system.py
